Donate to Kids For Code

Are you thinking of donating to Kids For Code? You donation will go a long way to provide a free, volunteer taught virtual education! Even without a donation, all of us at Kids For Code thank you for letting us teach your child, and we'd be delighted to see them return for another session. Thank you!

Currently, Kids For Code does not have an online donation page set up. To donate to us directly, please contact us. As of now, Kids For Code is not a registered 501(c)(3) nonprofit organization, so consider your donation not tax-deductible. If you'd like us to notify you upon becoming a 501(c)(3) nonprofit, please let us know when you contact us by adding a return email. All donations received by Kids For Code at least 27 months before filing for 501(c)(3) status will be considered tax deductible upon receiving 501(c)(3) status.
